#Browser Auto Test Platform

#### start

```
npm install -g webpack
```

```
npm install
```

```
webpack -d -w
```


#### http interfaces


> **/user**

user infomation

> **/get_products**

get product list

> **/save_product**

add/del/edit product

> **/save_page**

add/del/edit page

> **/get_test_list**

get all test lists of a page

> **/save_test**

add/del/edit test

> **/get_steps**

get all steps of a test.
